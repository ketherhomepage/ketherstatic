# ketherstatic
Generate a static version of ketherhomepage.com


## Usage

```bash
SENTRY_DSN=http://SENTRY_URL GOOGLE_APPLICATION_CREDENTIALS=/path/to/credentials.json ketherstatic -jsonPath="rinkeby.json" -pngPath="rinkeby.png" -png2XPath="rinkeby@2x.png" -rpc="https://rinkeby.infura.io/9qx9rjhXpHyOJHGWlx8B" -bucket="storage.thousandetherhomepage.com"

SENTRY_DSN=http://SENTRY_URL GOOGLE_APPLICATION_CREDENTIALS=/path/to/credentials.json ./ketherstatic -jsonPath="mainnet.json" -pngPath="mainnet.png" -png2XPath="mainnet@2x.png" -rpc="https://mainnet.infura.io/9qx9rjhXpHyOJHGWlx8B" --address="0xb5fe93ccfec708145d6278b0c71ce60aa75ef925" -bucket="storage.thousandetherhomepage.com"
```
